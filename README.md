This repo contains a set of instructions to upskill the developer on certain technologies.
The goal is to use AI to maximize the knowledge transfer from the model knowledge to the developer brain.

This means the user must do effort and make its brain work, the AI should just adapt its exercices to the developer level in the targeted technology.

It is structured in the following way : 

- ai.md 
- memory/
- sandbox/

The ai.md contains all the instructions that the agent must follow to give the user an optimal learning experience. The user should make sure your agent or IDE reads it systematically. Alternatively the user can create a skill to give the agent the right role.

The memory folder contains one file per technology studied (python, typescript, react, SQL, terraform, ...) that summarises the knowledge of the developer. Each user will have its local version, it must be assumed that all the memory elements are about the user chatting with the AI and that the user has not modified the memory files.

The sandbox contains exercices generated by AI to upskill the developer. Each exercice is created in a subfolder that can be deleted once the exercice has been completed.

As of March 2026 it is advised to use Claude models and the same one if possible to ensure consistency accross memory storage.

## YOU NEED TO DISABLE AUTOCOMPLETE
If you IDE has auto-complete, disable it. This repo is useless if you don't.
